🥐 PastryWorld
Un univers gourmand, lumineux et élégant
PastryWorld est un site vitrine imaginé comme une boutique de pâtisseries moderne :
un espace doux, aéré, où chaque création est présentée comme une petite œuvre d’art.
Le projet met l’accent sur la simplicité, la clarté et une atmosphère chaleureuse.

✨ Fonctionnalités
- Présentation élégante de pâtisseries et créations sucrées
- Mise en page fluide et responsive
- Visuels lumineux et cohérents
- Navigation simple et intuitive
- Structure HTML/CSS propre et accessible

🎨 Technologies utilisées
- HTML5
- CSS3
- JavaScript (léger, pour l’interactivité)

🧁 Philosophie du projet
PastryWorld a été conçu comme une petite parenthèse gourmande :
un lieu où l’on prend le temps de regarder, de savourer, de respirer.
Chaque section est pensée comme une vitrine, chaque image comme une invitation.
